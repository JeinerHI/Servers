Es de código abierto desarrollado por OpenLDAP Project y cuenta con las sgtes características:

- Multiplataforma: Compatible con múltiples sistemas operativos
- Código libre: Licencia de código abierto
- Basado en X.500
- Estructura DIT: Estructura de árbol denominada DIT
- Soporta IPv3 y LDAPv3
- Internacionalización: Uso de caracteres UTF-8
- Integración Linux
